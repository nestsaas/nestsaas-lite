# Welcome to NestSaaS

NestSaaS is a modern framework for building ai SaaS website with powerful tools.

Packed with powerful features like secure Authentication, Payment Processing, Blog functionality, SEO optimization, and customizable Themes, NestSaaS delivers everything you need to create a standout online presence.

Designed with extensibility at its core, scale, and succeed with ease.

## Key Features

- **Built-in Payment Processing** - Accept payments and manage subscriptions
- **User Management** - Comprehensive user roles and permissions
- **SEO Optimization** - Built-in tools for better search engine visibility
- **Modern Tech Stack** - Built with Next.js, React, Tailwind CSS, and Prisma
- **Responsive Design** - Beautiful UI that works on all devices

## Tech Stacks

NestSaaS uses Next.js App Router, Prisma, Shadcn UI, Auth.js, React Query, Zod, Lucide, Resend, and more.

- [Next.js](https://nextjs.org/): The React Framework for the Web
- [Prisma](https://www.prisma.io/): A best-in-class TypeScript ORM
- [PostgreSQL]: Neon or Supabase
- [Tailwind CSS](https://tailwindcss.com/): A utility-first CSS framework
- [Radix UI](https://www.radix-ui.com/): An open source component library 
- [Shadcn/UI](https://ui.shadcn.com/): A set of beautifully-designed, accessible components and a code distribution platform
- [Magic UI](https://magicui.design/): UI library built with React, Typescript, Tailwind CSS, and Motion. Perfect companion for shadcn/ui.
- [Tailark](https://tailark.com/): pre-built UI blocks designed for marketing websites.
- [Auth.js](https://authjs.dev/): Authentication for the Web.
- [Resend](http://resend.com/): Resend is the email API for developers.
- [Fumadocs](https://fumadocs.dev/): A beautiful documentation framework for Developers, flexible and performant, with everything from Next.js.
- [Stripe](https://stripe.com): Payments infrastructure for the internet
- [Content Collections](http://content-collections.dev/):  Type-safe data content collections
